# Instalador-Zabbix

chmod +x instalar_zabbix7.sh

./instalar_zabbix7.sh

Um script para instalar o Zabbix 7 LTS no DEBIAN 12

Versão install_zabbix_74.sh -> instala versão 7.4

UPGRADE do 7 para o 7.4 - faz o backup do banco de dados -> upgrade_zabbix_74.sh

Upgrade Zabbix 6.4.x → 7.0 → 7.4 - Debian 12 + MariaDB -> upgrade_zabbix_64_to_74.sh
